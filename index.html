jquery.freezeheader

A simple jquery plugin to freeze header row in html table.

Installation

Include script after the jQuery library (unless you are packaging scripts somehow else):

<script src="/path/to/jquery.freezeheader.js"></script>

Do not include the script directly from GitHub (http://raw.github.com/...). The file is being served as text/plain and as such being blocked in Internet Explorer on Windows 7 for instance (because of the wrong MIME type). Bottom line: GitHub is not a CDN.

Usage

Create a table with fixed header in the top browser:

        $(document).ready(function () {
            $("#tableid").freezeHeader();
        })

Create a table with fixed header and scroll bar:

        $(document).ready(function () {
            $("#tableid").freezeHeader({ 'height': '300px' });
        })
